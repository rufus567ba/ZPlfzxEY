# 微信智能招聘小程序

## 前言

随着移动互联网的快速发展，微信小程序因其便捷性、易用性受到了广大用户的喜爱。在此背景下，我们开发了一款基于SSM框架和微信小程序的智能招聘小程序，致力于为企业提供高效的招聘服务，为求职者提供便捷的求职体验。

## 内容介绍

微信智能招聘小程序主要包括以下功能：职位搜索、简历投递、面试邀请、职位推荐等。企业用户可以发布职位，筛选简历，邀请求职者面试；求职者可以查看职位信息，投递简历，接收面试邀请。此外，我们还采用了智能匹配算法，为用户提供更精准的职位推荐。

## 技术介绍

- 语言：Java
- 使用框架：Spring、Springmvc、Mybatis、微信小程序
- 前端技术：JS、Vue、CSS3、Uniapp
- 开发工具：IDEA/Eclipse、Uniapp
- 数据库：MySQL 5.7/8.0
- 数据库管理工具：phpstudy/Navicat
- JDK版本：jdk1.8
- Maven：apache-maven 3.8.1-bin
- 前端环境：Node.Js 12\14\16
